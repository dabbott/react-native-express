JavaScript 语言每年都有更新和新特性，如果你有一段时间没有使用 JavaScript 了，可能会对 React Native 里使用的语法感到不熟悉。

> 如果你是 JavaScript 的新手，想在开始 React Native 之前学习一下 JavaScript，请参考原作者写的 [JavaScript 速成教程](https://www.javascript.express)，其内容涵盖了主要的 JavaScript 知识点，包括 TypeScript。

## 关键新特性

在本章节中，我们将介绍近年来在 React Native 应用程序中常见的一些关键新特性：

- 导入和导出
- 块作用域变量声明
- 箭头函数
- 类
- 解构赋值
- 展开语法
- Async/await

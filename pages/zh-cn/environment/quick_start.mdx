在本节中，我们将使用 `expo-cli` 来建立一个新的 React Native 应用程序。

`expo-cli` 需要依赖 node.js v10 或更高的版本。如果你的本地环境尚未安装 node.js，请访问 [node.js](https://nodejs.org/en/) 官网，依照网站的教程下载适合你本地的安装包，并安装 node.js。

## 安装

首先，运行以下指令来安装 `expo-cli` 命令行工具

```bash
npm install --global expo-cli
```

然后就可以调用 `expo-cli` 来创建一个新的应用程式

```bash
expo init MyApp --template blank

cd MyApp
npm start
```

`expo-cli` 可能需要几分钟的时间来下载依赖包。

> 如果你已经安装了 `yarn` , Expo 会问你想用 `yarn` 还是 `npm`，两种方法的执行结果是一样的。

上述命令执行结束后，你将会看到一个 QR 码，如下图所示

## 更改代码

请尝试更改 `App.js` 中的代码，然后保存文件。你会看到修改的内容会自动在设备上生效。

# Navigating and Routes

- Every navigator supports different ways of navigating
  - Stack: `push`
  - Tabs: `navigate`
  - Drawer: `openDrawer`
- When navigating, we specify a screen name and params
  - `navigator.push("Screen2", { paramA: "Hello!" })`
